Coloque aqui o seu ícone do aplicativo como "icon.ico".

Requisitos recomendados:
- Formato: .ico
- Tamanho: 256x256 (contendo múltiplos sizes dentro do .ico é ainda melhor: 16, 32, 48, 64, 128, 256)
- Caminho final: build/icon.ico

Dica: se você tiver um PNG 1024x1024, use um conversor (como IcoFX, GIMP com plugin, ou sites online) para gerar o .ico.
